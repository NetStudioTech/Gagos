# GAGOS Documentation

Welcome to GAGOS - Lightweight DevOps Platform.

GAGOS is a comprehensive DevOps administration platform with network diagnostics, Kubernetes management, CI/CD pipelines, database tools, and more - all in a single container.

## Getting Started

- [Installation](installation.md) - Docker, Kubernetes, and source installation
- [Architecture](ARCHITECTURE.md) - System design and components

## Features

### Infrastructure
- [Network Tools](features/network.md) - Ping, DNS, traceroute, SSL checks, and more
- [Kubernetes Management](features/kubernetes.md) - Full K8s resource management
- [Monitoring](features/monitoring.md) - Cluster resource monitoring

### CI/CD
- [CI/CD Pipelines](CICD_USER_GUIDE.md) - Kubernetes pipelines and freestyle jobs

### Data Services
- [Database Tools](features/database.md) - PostgreSQL, MySQL, Redis clients
- [Elasticsearch](features/elasticsearch.md) - Cluster management and queries
- [S3 Storage](features/s3.md) - S3-compatible object storage

### Utilities
- [Developer Tools](features/devtools.md) - Base64, hashing, certificates, SSH keys
- [Web Terminal](features/terminal.md) - Browser-based shell access
- [Notepad](features/notepad.md) - Multi-tab text editor
